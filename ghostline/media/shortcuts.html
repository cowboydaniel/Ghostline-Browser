Tabs & Windows

New Tab Open a new tab in the current window
Ctrl + T
New Window Open a new browser window
Ctrl + N
New Private Window Open a new private browsing window
Ctrl + Shift + N
Close Tab Close the current tab
Ctrl + W
Close Window Close the entire browser window
Ctrl + Shift + W
Reopen Closed Tab Restore the last closed tab
Ctrl + Shift + T

Navigation

Go to Address Bar Focus the address bar to enter a URL or search
Ctrl + L
Back Go to the previous page
Alt + ←
Forward Go to the next page
Alt + →
Reload Page Refresh the current page
Ctrl + R
Hard Reload Refresh bypassing cached content
Ctrl + Shift + R

Page

Find in Page Search for text on the current page
Ctrl + F
Print Print the current page
Ctrl + P
Save Page Save the current page to disk
Ctrl + S
Zoom In Increase the page zoom level
Ctrl + +
Zoom Out Decrease the page zoom level
Ctrl + -
Reset Zoom Reset zoom to default level
Ctrl + 0

Privacy & Settings

Open Settings Access browser settings and preferences
Ctrl + ,
Privacy Dashboard View and manage your privacy settings
Ctrl + P
Clear Browsing Data Open the clear browsing data dialog
Ctrl + Shift + Delete

Developer Tools

Open Developer Tools Access the browser's developer console and inspector
Ctrl + Shift + I
View Page Source View the HTML source of the current page
Ctrl + U
Inspect Element Inspect elements on the page
Ctrl + Shift + C
